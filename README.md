# README
## MyPrettyJson

+ 快捷键 shift + cmd + j 格式化.
+ 快捷键 shift + cmd + u 压缩.
+ 快捷键 shift + cmd + r 美化，去掉json套json的转义
   key 加上 `[⭕️]` 表示原来是 字符串， 数组中  `[⭕️]` 表示下一个元素原来是字符串

+ 快捷键 shift + cmd + e 还原结构，json 套json

